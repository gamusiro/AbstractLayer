# 概要
Vulkan, DirectX12を利用してゲームを作ってみました。
入力制御にはXInputとWindowsAPIを利用しています
オーディオ制御はXAudio2を利用
オンライン対戦にはPhotonを利用しています

## cmake
- 1.CMakeListsのディレクトリまで移動する
- 2.buildフォルダを作成
- 3.ソリューション作成
```
   cd ../AbstractLayer-main
   mkdir build
   cd build
   cmake..
```
